# Page d’accueil e-commerce (Maquette → Intégration responsive)
<img width="1437" height="935" alt="Capture d’écran du 2025-09-22 21-51-49" src="https://github.com/user-attachments/assets/880429ea-145e-499e-9120-e063f54bb3d5" />

Ce projet consiste à reproduire fidèlement la maquette fournie pour la page d’accueil du site Nature Emoi, boutique en ligne spécialisée dans les plantes d’intérieur.
L’intégration est réalisée en HTML sémantique, CSS (mobile-first), avec Flexbox, CSS Grid et de bonnes pratiques de structure.
